# 🌟 갓생 루틴 플래너 (AI 기반 맞춤형 일정 앱)

> **"AI가 나만의 루틴을 만들어주는 플래너"**  
> 대화 기반 맞춤형 목표 추천과 일정 시각화를 통해 갓생을 돕는 스마트 플래너입니다.

---

## 🚀 주요 기능

### 🧠 맞춤형 목표 추천

- 사용자의 대화 내용을 분석해 **AI가 개인별 루틴을 자동 생성**
- “요즘 피곤해” → 스트레칭, 조깅, 독서 등의 **맞춤형 활동 제안**

### 📅 스케줄 관리

- **하루 / 주간 일정 등록 및 시각화**
- **일정의 진행 상태(과거 / 현재 / 예정)** 에 따라 **UI 색상 자동 변경**
